# Core Services Overview

The Activiti engine provides six core services that form the foundation of process management:

## 📚 Available Services

| Service | Description | Status |
|---------|-------------|--------|
| [Repository Service](./repository-service) | Manage process definitions and deployments | ✅ Available |
| [Runtime Service](./runtime-service) | Execute and control process instances | ✅ Available |
| [Task Service](./task-service) | Manage user tasks and assignments | ✅ Available |
| [History Service](./history-service) | Query historical process data | ✅ Available |
| [Management Service](./management-service) | Administer engine operations | ✅ Available |
| [External Task Service](./external-task-service) | External task worker pattern | ✅ Available |

## 📖 Learning Path

1. Start with [Repository Service](./repository-service) - Deploy your first process
2. Move to [Runtime Service](./runtime-service) - Execute processes
3. Use [Task Service](./task-service) - Handle user interactions
4. Check [History Service](./history-service) - Analyze execution data
5. Explore [Management Service](./management-service) - Monitor engine health
6. Try [External Task Service](./external-task-service) - Build distributed systems
