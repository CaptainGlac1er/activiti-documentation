# Examples

This section contains fully working examples that demonstrate common Activiti workflow patterns.

## Available Examples

| Example | Description | Files |
|---------|-------------|-------|

## How to Use These Examples

Each example includes:

1. **Summary** — Overview of the workflow and what it demonstrates
2. **Main Process** — The top-level BPMN process
3. **Sub-Processes** — Individual child processes
4. **Service Delegates** — Java implementation classes
5. **Process Extensions** — Variable mappings and configuration
6. **REST API** — Controller endpoints for triggering workflows
